# Copy Review Checklist

Complete evaluation criteria for product page copywriting.

---

## Hero Section

### Headline
| Criteria | Weight | Check |
|----------|--------|-------|
| Communicates outcome (not feature) | HIGH | [ ] |
| Under 10 words | MED | [ ] |
| Contains benefit or transformation | HIGH | [ ] |
| Specific (not vague) | HIGH | [ ] |
| No jargon without context | MED | [ ] |

**Good:** "Cut WHMCS Admin Time in Half with AI"
**Bad:** "AI-Powered WHMCS Integration Module"

### Subheadline
| Criteria | Weight | Check |
|----------|--------|-------|
| Expands on headline promise | HIGH | [ ] |
| Adds specificity or mechanism | MED | [ ] |
| 1-2 sentences max | LOW | [ ] |
| Creates curiosity or urgency | MED | [ ] |

### Hero CTA
| Criteria | Weight | Check |
|----------|--------|-------|
| Action verb (Get, Start, Unlock) | HIGH | [ ] |
| Indicates what user receives | HIGH | [ ] |
| Visible without scrolling | HIGH | [ ] |
| Price shown (if applicable) | MED | [ ] |
| Contrasting color | MED | [ ] |

## Value Proposition

### Benefits vs Features
| Criteria | Weight | Check |
|----------|--------|-------|
| Benefits outnumber features | HIGH | [ ] |
| Each feature linked to benefit | HIGH | [ ] |
| Outcomes are quantified | MED | [ ] |
| User is the subject ("You get...") | MED | [ ] |

**Feature:** "28 AI tools"
**Benefit:** "Automate 28 different WHMCS operations without writing code"

### Specificity
| Criteria | Weight | Check |
|----------|--------|-------|
| Numbers instead of vague words | HIGH | [ ] |
| Time savings quantified | MED | [ ] |
| Results/outcomes specific | HIGH | [ ] |
| Comparisons are concrete | MED | [ ] |

## Scoring Guide

| Score | Meaning |
|-------|---------|
| 9-10 | Excellent - Ready to ship |
| 7-8 | Good - Minor improvements needed |
| 5-6 | Average - Several issues to address |
| 3-4 | Needs work - Major revision required |
| 1-2 | Poor - Complete rewrite recommended |

### Priority Levels

- **HIGH**: Fix before launch - directly impacts conversion
- **MED**: Fix soon - noticeable improvement expected
- **LOW**: Nice to have - polish and refinement
